# 修复视频带水印问题

## 任务描述
解析出的视频带水印，需要改为提取无水印链接。

## 原因分析
- 分享页返回的 `play_addr` URL 包含 `/playwm/`（play with watermark），即带水印版本
- 分享页不包含 `bit_rate` 数据，无法从中获取无水印地址
- 抖音 CDN 机制：`/playwm/` → 带水印，`/play/` → 无水印

## 待办
- [x] 分析分享页实际返回的数据结构
- [x] 确认 `bit_rate` 在分享页中不存在
- [x] 实现 URL 中 `playwm` → `play` 的替换去水印
- [x] 验证构建通过

## 修改文件
- `src/lib/parser.ts` - 添加 `removeWatermark` 函数，将 URL 中 `/playwm/` 替换为 `/play/`

## 结果
- 视频 URL 从 `https://...aweme/v1/playwm/?...` 变为 `https://...aweme/v1/play/?...`
- 前端零影响，数据结构不变
